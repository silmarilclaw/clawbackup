**Drop any files here that you want Link to access!**

This folder is connected to Link's workspace, so anything you put here will be instantly available to me.

## ✅ Supported File Types

- **Documents:** PDF, Word, Text, Markdown, etc.
- **Images:** JPG, PNG, GIF, WebP
- **Data:** CSV, JSON, Excel, etc.
- **Code:** Any programming language
- **Archives:** ZIP, TAR, etc.

## 🔗 How It Works

- This folder: `~/Documents/OpenClaw/`
- Linked to: `/Users/silclaw/.openclaw/workspace/shared`
- I can access files with: `shared/filename.ext`

1. Drop a PDF here → I can read and analyze it
2. Drop an image → I can describe or process it
