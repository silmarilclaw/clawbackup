## Key Lessons
- `config.patch` on `agents.list` **replaces the entire array** — always include all agents
- Telegram `channels.telegram.accounts` can hold stale bot tokens that interfere
- Session stores with references to non-existent .jsonl files cause path validation errors
- Always verify full config state after patches — hidden nested keys can survive
